### Kurulum

Terminal üzerinde aşağıdaki komutlar çalıştırılmalıdır: 
```sh
git clone https://github.com/aslihanturkdonmez/tarifka.git
cd tarifka
npm install
```

IOS için
```sh
cd ios && pod install
cd ..
react-native run-ios
```

Android için
```sh
react-native run-android
```
